The ConvLayer function returns a the weights, the biases and the output of a convolutional layer and the shape of its output.

Parameters
layerIn: The input of the layer
inputShape: The shape of the input of the layer. This must be a list with four dimensions : [ batch, channels, height, width ]
filters: The number of filters
fHeight: The filters' height
fWidth: The filters' width
vStride: The convolution vertical stride
hStride: The convolution horizontal stride
inPad: The type of padding to apply. It must be one of the value enumarated in ConvNet.Paddings, an int or a pair of int
act: The activation function of the layer. Is must be one of the value enumarated in 'ActivationFunctions'
name: The name of the layer
clipMin: The minimum value allowed for the gradient compoments associated to this layer. If None (default), the clipping is disabled
clipMax: The maximum value allowed for the gradient compoments associated to this layer. If None (default), the clipping is disabled
Returns
A tuple containing the newly created parameters, the output and its shape : (w, b, y, shape)

Parameters
layerIn: The input of the layer
inputShape: A list containing the dimensions of the input in the form [ batch, d0, d1, ..., dn ]
filters: The number of filters
act: The activation function. This must be one of the value enumerated in Layer.ActivationFunctions
name: The name of the layer
clipMin: The minimum value allowed for the gradient compoments associated to this layer. If None (default), the clipping is disabled
clipMax: The maximum value allowed for the gradient compoments associated to this layer. If None (default), the clipping is disabled
Returns
A tuple containing the newly created variables, the output and its shape : (w, b, y, shape)

name: The name of the variable
method initArg

InitMethod.normal A tuple containing the mean and the standard deviation
