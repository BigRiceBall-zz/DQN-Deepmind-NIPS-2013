Detailed Description

The ImageSet class implement an image container.

The ImageSet class expects that the images to store are numpy arrays of np.float32 of shape [h, w] where h is the height of the image and w its width.

If the set if full, this class pre-allocate an array of N images before storing new images.

The images stored in the set are actually copied
