def DQN-Deepmind-NIPS-2013.Plotter.Plotter.addPercentile (   self,
  group,
  name,
  percentagesPairs 
)

The addPercentile methods add a percentile chart to the list.

Parameters
group: The name of the group the percentile chart belongs to
name: The name of the chart
percentagesPairs: A list of tuple of values between and including 0 and 100. If percentagesPairs is made of pairs (p_i, p_j) the chart will be made of :
  • A line at (x, percentile(A(x), p_i))
  • A line at (x, percentile(A(x), p_j))
  • An area filled between the two previous lines with color where the alpha value is proportional to p_i - p_j

The addPlot method adds a plot to the given group.

Parameters
group: The name of the group the plot belongs to
name: The name of the new plot to add. This name should be unique inside the given group
cnt: How many lines will be plotted at the same time
styles: The style of the lines to plot. Styles must have 'cnt' elements. The elements in 'styles' will be directly passed to 'pyqtgraph.mkPen'. See pyqtgraph.mkPen documentaiton for more information
