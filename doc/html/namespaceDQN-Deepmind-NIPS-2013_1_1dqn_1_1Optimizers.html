The RMSProp function implements the RMSProp algorithm.

Given:

  • \(f(\boldsymbol{\theta})\) : the function to minimize
  • \(\boldsymbol{\theta}_t = (\theta_{t,0}, ..., \theta_{t,n-1})\) : the value of the parameters at the time step \(t\) that we want to update to minimize \(f\)
  • \(\boldsymbol{\theta}_0\) : the initial value of the \(\theta\) parameters
  • \(\boldsymbol{m}_t = (m_{t,0}, ..., m_{t,n-1})\) : An array of size \(n\) such that \(m_{0,i} = 0\,\forall i \in \{0, ..., n - 1\}\)
  • \(\boldsymbol{u}_t = (u_{t,0}, ..., u_{t,n-1})\) : An array of size \(n\) such that \(u_{0,i} = 0\,\forall i \in \{0, ..., n - 1\}\)
  • \(\lambda\) : The learning rate
  • \(\mu\) : The momentum
  • \(\delta\) : The decay
  • \(\epsilon\) : A small number to avoid dividing by zero

At each iteration, the RMSProp algorithm performs the following updates

  1. \(\boldsymbol{g}_t = \boldsymbol{\nabla_\theta} f(\boldsymbol{\theta})|_{\boldsymbol{\theta}= \boldsymbol{\theta_{t-1}}}\)
  2. \(m_{t,i} = \delta\,m_{t-1,i} + (1-\delta) g_{t,i}^2\quad\quad \forall i \in \{0, ..., n - 1\}\)
  3. \(u_{t,i} = \mu\,u_{t-1, i} + \lambda \frac{g_{t,i}} {\sqrt{m_{t,i} + \epsilon}}\quad\quad \forall i \in \{0, ..., n - 1\}\)
  4. \(\boldsymbol{\theta}_t = \boldsymbol{\theta}_{t-1} + \boldsymbol{u}_t\)
Parameters
grads: A list of the component of the gradient of the function to minimize derived with respect to the parameters to modify
params: A list of parameters to update
learning_rate: The learning rate
momentum: The momentum
decay: The decay
epsilon: A small value to avoid dividing by zero
Returns
A list of updates operation to pass to a theano function in order to apply RMSProp algorithm
